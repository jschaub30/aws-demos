About this project

AskJerm is a Generative AI implementation using AWS Bedrock. The AskJerm agent invokes a RAG (Response-Augmented Generation) pipeline to combine queries with knowledge bases that have been ingested and stored in a vector database. The augmented query is then sent to the Anthropic Claude v2 foundational model.
